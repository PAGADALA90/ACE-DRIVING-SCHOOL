# 🚗 ACE Driving School Management System

## 📌 Project Overview

The **ACE Driving School Management System** is a mainframe-based application developed to automate the management of a driving school. The system provides facilities for maintaining student and instructor records, updating information, generating reports, and managing data efficiently using IBM mainframe technologies.

---

## 🛠️ Technologies Used

* **COBOL**
* **CICS COBOL**
* **DB2**
* **VSAM KSDS**
* **JCL**
* **BMS Maps**

---

## ✨ Features

* Student Record Management
* Instructor Record Management
* Add New Records
* Update Existing Records
* Delete Records
* Search Records
* Input Validation
* User-Friendly CICS Screens
* Report Generation
* DB2 and VSAM Data Storage

## 🔄 System Flow

1. User accesses the Main Menu.
2. Selects Student or Instructor Maintenance.
3. Performs Add, Update, Delete, or Inquiry operations.
4. Data is validated and stored in DB2/VSAM.
5. Reports are generated based on stored records.
